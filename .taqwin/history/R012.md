## Changes Made

### Security Features
- **Origin Validation**: Strict origin checking with pattern matching for localhost
- **Rate Limiting**: Per-IP rate limiting (100 requests/60s), automatic IP blocking
- **Message Validation**: JSON validation, schema validation, type validation, size limits
- **Message Sanitization**: Script tag removal, string length limits, key sanitization
- **IP Blocking**: Automatic blocking for violations, 5-minute block duration
- **Security Events**: Emission of security events to event emitter

### Security Components
- RateLimiter: Sliding window implementation, remaining requests tracking
- OriginValidator: Exact match + pattern matching for localhost
- MessageValidator: Pydantic schema validation, content sanitization
- WebSocketSecurity: Combined manager integrating all components
