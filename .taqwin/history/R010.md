## Changes Made

### ConnectionManager Features
- **Heartbeat**: Server-side ping every 30s, pong timeout detection (60s)
- **Health Monitoring**: Tracks connected_at, last_pong, last_message, message_count, error_count, health_score
- **Message Queuing**: Queues messages for disconnected sessions, max 1000, age limit 1 hour
- **Backpressure**: Rate limiting per connection (default 100 msg/s)
- **Message Batching**: 50ms batch window, automatic flush
- **Acknowledgment**: Sequence numbers for all messages
- **Auto-Cleanup**: Idle timeout (5 minutes), stale queue cleanup (every minute)

### Router Updates
- Uses get_connection_manager() singleton
- Flushes queue on reconnection
- Handles pong messages
- Handles ACK messages
- Records errors on JSON decode
- Emits error events on exceptions
- Uses sequence numbering in broadcast_event
