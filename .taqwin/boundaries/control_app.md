# Boundary: Control App

Stamped: 2026-02-10T01:00+05:30

## Owns
- Visualization and operator controls
- Execution triggers and UX failure visibility
- Rendering performance (large logs, ticket graphs, memory views)

## Does Not Own
- Memory truth
- Logic already present in TAQWIN or KNEZ

## Access Discipline
- Read-only on memory by default.
- Any write capability requires explicit, logged grant.
